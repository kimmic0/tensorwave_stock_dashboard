# tensorwave_stock_dashboard
Create a React application (Next.js).  Utilized the AlphaVantage API, which provides stock market related information and data.

- Displays 15 stocks (tickers)
- If a user clicks on any one of the stocks (tickers) on homepage, they are routed  to the stock details page.
- Shows the company symbol, asset type, name, description, exchange, sector, industry and market capitalization. If any of this information is not available then display “N/A” instead
- Shows a list of historical prices (time_series_daily)
- application responsive (mobile friendly)
- contains logos for all the companies/stocks and display them on the homepage as well as the stock-details page
